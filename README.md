### 简介

本项目将介绍常见的线性、树形、图状数据结构，并使用Python等进行实现。

在开始阅读本项目之前，请先阅读[基本概念](https://github.com/tim-chow/DataStructure/tree/master/%E5%9F%BA%E6%9C%AC%E6%A6%82%E5%BF%B5)。
